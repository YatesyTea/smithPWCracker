Detailed Description

Definition of a datatype for a list of candidate solutions.

Datatype to hold a collection of candidate solutions in the form of an array along with a variable that tells us the array index of the last entry in the list.

NOTE that it is intended that all add/remove actions are done via the provided functions so that the internal counter of the numer of entries is kept correct
